# Money 2.4.1

版本号：`2.4.1`

versionCode：`106`

## 主要变化

- 净资产目标融合到分析页的“期末净资产”主卡，集中展示目标金额、实际完成比例、进度条和剩余或超出金额。
- 浏览历史月份时，目标进度使用所选月份的期末净资产计算，避免把当前资产错误套入历史月份。
- 账户页移除顶部整体资产与目标展示，页面聚焦正常、隐藏和关闭账户本身。
- 净资产目标卡保持纯展示；设置、修改和清除目标仍通过独立管理入口完成。

## 数据与兼容性

- 数据库继续使用 Room v14，没有新增数据库迁移。
- 金额继续使用最小货币单位 `Long`，目标比例计算避免长整型乘法溢出。
- 备份格式保持明文 JSON v4，与 2.4.0 完全兼容。

## 验证

- 全量单元测试与 Debug Lint 通过。
- Debug、Android 测试源码和 Release APK 编译通过。
- Release APK 使用预期正式证书完成签名验证。
