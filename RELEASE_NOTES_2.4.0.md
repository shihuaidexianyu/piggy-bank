# Money 2.4.0

版本号：`2.4.0`  
versionCode：`105`

## 主要变化

- 一级导航统一为首页、明细、分析和账户，全局“记一笔”使用居中的收入、支出、转账、校对四入口。
- 首页突出净资产和本月收支，压缩预算入口，并仅在确有待办时显示提醒与待核对账户。
- 明细页改为带吸顶日期和可见筛选条件的账单式时间线。
- 分析页保留资金流动关系图，并提供可选择日期的每日收入、支出和累计净流图；坐标明确标记正、零、负方向。
- 账户页按正常、隐藏和关闭状态分组；净资产目标采用与账户一致的颜色进度卡并置于总资产下方。
- 生物识别锁在进入锁定状态时自动唤起；取消或失败后保持锁定并允许手动重试，不会循环弹窗。
- 账户顺序管理移入设置页，设置按显示、隐私、通知、数据和关于重新分组。

## 数据与兼容性

- 数据库保持 Room v14，可从 2.0.2/DB13 直接迁移，不使用破坏性迁移。
- 保留固定 reconciliation delta、最小货币单位 `Long` 和四类账本事件语义。
- 备份继续使用明文 JSON v4；设备本地主题、生物识别和遮罩设置不会被导入覆盖。

## 验证

- 单元测试、Lint、Debug 构建和 Android 测试 APK 编译通过。
- Release APK 使用预期证书完成 v2 签名验证。
