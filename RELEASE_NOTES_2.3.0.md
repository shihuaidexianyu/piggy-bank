# Money 2.3.0 发布说明

- 版本号：`2.3.0`
- versionCode：`103`
- 数据库：Room v14
- 备份格式：Money backup v4（明文 JSON）

## 本次完成

- 数据安全：Room v14、全量无损迁移链、四类流水软删除、`operationId` 幂等写入、半开时间区间和固定核对差额。
- 账户生命周期：隐藏、关闭、重新开启和迁移问题提示；关闭账户保持历史统计不变。
- 备份恢复：v1-v4 显式迁移、32 MiB/20 万记录限制、缓存文件哈希绑定、事务导入、安全快照、持久回执和导入撤销。
- 提醒与通知：移除精确闹钟权限，统一为 WorkManager 周期检查与去抖同步，支持记账、跳过、编辑及并发去重。
- 隐私：生物识别状态机、后台重锁、最近任务保护、Widget/通知金额独立遮罩以及锁后深链恢复。
- 日常记账：首页/明细/分析/账户四个一级页面，自适应底栏或侧栏，全局记账 FAB，草稿保存和防重复提交。
- 首页与分析：本月净资产、收入、支出、净现金流、月预算、每日趋势、账户流向、转账路径和精确明细下钻。
- 账户与目标：正常/隐藏/关闭分组、账户详情快捷动作，以及单一净资产储蓄目标。
- 可用性：统一加载/空态/错误态、全局撤销、关键交互 48dp、图表无障碍摘要和大字号可滚动布局。
- 一致性与效率：首页、分析、账户、Widget 和备份使用一致性快照；导出采用流式 JSON 编码。
